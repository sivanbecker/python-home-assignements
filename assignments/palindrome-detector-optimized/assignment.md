## Assignment Topic: Palindrome Detector (Optimized)

Write a script that checks if a given string (or phrase) is a palindrome.

Requirements:
- Ignore spaces, capitalization, and punctuation.
- Example: "A man, a plan, a canal: Panama" should return True.
- **Memory constraint: ≤ 500KB**
- String length: up to ~1,000,000 characters
